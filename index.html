Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipilação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento Web como o Streamlit e o Github, foram usadas para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na Índia.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link Web. O painel pode ser acessado por qualquer dispositivo conectado a internet.

As ferramentas utilizadas foram:

  • Git, Github.
  • Python, Pandas, Plotly.
  • Jupyter Notebook.
  • Streamlit e Streamlit Cloud.
  • Linux.
  • Terminal.
  • Vi/Vim
  • Anaconda Environment
